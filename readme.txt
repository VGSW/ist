SYNOPSIS

    make clean
    make test
    make

DESCRIPTION

    Simply run `make` in this directory. Make will download the necessary files and start bottle. As soon as bottle is running one can access it locally: http://127.0.0.1:8080

    commandline invocation
        python3 main.py --sort=desc --sort_col=country > output.html

    get usage
        python3 main.py --help

    the files data.csv and data.sqlite3 are expected in the same directory as main.py
